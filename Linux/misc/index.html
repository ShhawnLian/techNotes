    将 RMySQL 源文件复制到服务器上，

    R CMD INSTALL RMySQL_0.10.14.tar.gz

    centos 6.5 上源码安装 MariaDB 5.5¶

    参考 1. Generic Build Instructions

    step 1¶

    mkdir build-mariadb
cd build-mariadb

    step 2¶

    注意设置 prefix

    参考 What is cmake equivalent of ‘configure –prefix=DIR && make all install ‘?

    cmake ../mariadb-5.*.* -DCMAKE_INSTALL_PREFIX:PATH=~/mariadb

    step 3¶

    make 
make install
